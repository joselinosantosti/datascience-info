PROJETOS

Sistema de recomendação

Desenvolvido inteiramente com Python, é baseado em filtragem colaborativa, na qual um conjunto de usuários dão notas para diversos itens. A partir do cálculo da distância euclidiana o algoritmo consegue identificar os usuários mais similares.

Sistema de busca similar ao Google, Bing etc

Um sistema desenvolvido inteiramente com Python e bibliotecas para crawler, preprocessamento. A indexação é feita com o SGBD MySQL.

Sistema de Automação residencial

Sistema desenvolvido no Raspberry Pi3 com Python, Flask e Firebase. Está sendo desenvolvido ainda um App Android que será integrado para controle de qualquer lugar do planeta.

Análise de dados PowerBI

Projeto com modelagem física e lógica do banco de dados, carregamento e análise dos dados e construção do Dashboard analítico.
